# 📝 To-Do List App
Keep your tasks organized! Add, edit, and mark tasks as done with this simple and interactive app built using HTML, CSS & JavaScript.

## ✨ Features
* Add new tasks
* Mark tasks as completed
* Delete tasks

## 🛠️ Built With
* HTML
* CSS
* JavaScript

## 🌱 Future Enhancements

🖊️ Edit task option
