# 📝 To-Do List App

A simple To-Do List web app built using HTML, CSS, and JavaScript.

## ✨ Features

- Add tasks
- Delete tasks
- Add task using Enter key
- Prevent empty tasks
- Clean and simple UI

## 🛠️ Built With

- HTML
- CSS
- JavaScript (DOM manipulation, Event Delegation)

## ✨ Future Improvements

- Add local storage (save tasks)
- Add edit option
- Add task completion (checkbox)
